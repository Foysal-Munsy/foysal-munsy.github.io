# Project cover images (no longer used)

The projects list is now **text-only** — clean rows with a title, summary, and a
dot-separated tech line. No cover images are rendered anywhere, so adding a
project is just an `.md` file under `src/content/projects/`. Nothing to size,
crop, or manage visually.

The `cover` / `coverAlt` / `coverBg` frontmatter fields are still accepted (so
older entries don't error) but are ignored. You can safely omit them.
